# webchat
        這是一個使用Socket.IO與Node.js製作的網頁聊天室，主要提供即時的訊息回應
安裝方法
=
        安裝node.js https://nodejs.org/zh-cn/
        安裝ngrok https://ngrok.com/

使用步驟
=
        1.打開cmd
        2.透過cd C:\你的電腦路徑\join chat移動到join chat文件夾下
        3.輸入node a.js啟動程式
        4.打開ngrok
        5.輸入 ngrok http 3000，建立伺服器
        6.在瀏覽器輸入http://localhost:3000/
